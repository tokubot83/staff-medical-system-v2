# Phase 2 実装計画書

## 📅 概要
- **開始予定**: Phase 1本番デプロイ後
- **期間**: 2-3週間
- **目的**: ユーザビリティの向上と段階的選択フローの実装

## 🎯 Phase 2の目標

### 1. 3段階選択フローの実装
現在の直接選択方式から、より直感的な段階的選択へ：

```
Step 1: 面談分類を選択
  ↓
Step 2: 面談種別を選択
  ↓
Step 3: カテゴリを選択（必要な場合）
  ↓
Step 4: 職員・日時を選択
```

### 2. UI/UXの改善項目

#### 2.1 視覚的フィードバック
- プログレスバーの実装
- 選択内容のサマリー表示
- 戻る/進むボタンの最適化

#### 2.2 選択支援機能
- 各面談種別の説明文追加
- カテゴリ選択時のガイダンス
- 推奨面談タイプの提案機能

#### 2.3 モバイル対応
- レスポンシブデザインの改善
- タッチ操作の最適化
- スワイプによるナビゲーション

## 🔧 技術仕様

### フロントエンド
```typescript
interface InterviewFlowState {
  currentStep: 1 | 2 | 3 | 4;
  selectedClassification?: 'regular' | 'special' | 'support';
  selectedType?: InterviewType;
  selectedCategory?: InterviewCategory;
  selectedStaff?: string;
  selectedDateTime?: Date;
}
```

### コンポーネント構成
```
<InterviewFlowContainer>
  <ProgressIndicator />
  <StepContent>
    {step === 1 && <ClassificationSelector />}
    {step === 2 && <TypeSelector />}
    {step === 3 && <CategorySelector />}
    {step === 4 && <StaffDateTimeSelector />}
  </StepContent>
  <NavigationButtons />
  <SelectionSummary />
</InterviewFlowContainer>
```
